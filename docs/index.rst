FastForward HTTP Client
=======================

**FastForward HTTP Client** is a PSR-11 compatible service provider that registers a fully functional set of PSR-18 HTTP client services using Symfony HttpClient and Psr18Client. It is designed to work seamlessly with the `php-fast-forward/container` autowiring system, providing a robust and flexible HTTP client integration for PHP applications.
