Dependencies
============

This package depends on:

- `symfony/http-client <https://github.com/symfony/http-client>`_
- `psr/http-client <https://www.php-fig.org/psr/psr-18/>`_
- `psr/http-message <https://www.php-fig.org/psr/psr-7/>`_
- `fast-forward/container <https://github.com/php-fast-forward/container>`_

All dependencies are managed via Composer and will be installed automatically.
