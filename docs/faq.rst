FAQ
===

**Q: What does this package provide?**

A: It registers a fully functional PSR-18 HTTP client using Symfony HttpClient and Psr18Client, with PSR-11 compatible service provider integration.

**Q: What are the requirements?**

A: PHP ^8.3 and the FastForward container for autowiring.

**Q: How do I get the PSR-18 client?**

A: Retrieve ``Psr\Http\Client\ClientInterface`` from your container.

**Q: Can I use the Symfony HttpClient directly?**

A: Yes, retrieve ``Symfony\Component\HttpClient\HttpClient`` from your container.

**Q: How do I contribute?**

A: Open an issue or submit a pull request on GitHub.
